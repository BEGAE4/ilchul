# 지도 정보 외부 API 정보 정리

# 1. Kakao Local API

카카오 로컬 API는 주소 ↔ 좌표 변환(지오코딩), 장소 검색(키워드, 카테고리) 기능을 제공합니다.

## 1-1. 주소로 좌표 변환

**축약 설명**

입력한 주소를 지도에 표시할 수 있도록 좌표와 주소 상세 정보를 제공합니다.

**제공 정보**

- 검색 결과 개수
- 페이지 조회 가능 개수
- 마지막 페이지 여부
- 입력 주소와 매칭된 전체 주소
- 주소 유형
- 좌표 정보
    - 경도
    - 위도
- 지번 주소 정보
    - 시/도
    - 시/군/구
    - 읍/면/동
    - 행정동
    - 법정동 코드
    - 행정동 코드
    - 산 여부
    - 지번 본번
    - 지번 부번
- 도로명 주소 정보
    - 도로명
    - 지하 여부
    - 건물 본번
    - 건물 부번
    - 건물명
    - 우편번호

## 1-2. 좌표로 행정구역정보 변환

**축약 설명**

좌표가 속한 법정동/행정동 정보를 조회합니다.

**제공 정보**

- 검색 결과 개수
- 행정동/법정동 구분
- 전체 행정구역 주소명
- 시/도
- 시/군/구
- 읍/면/동
- 리 또는 하위 행정구역
- 행정구역 코드
- 행정구역 중심 좌표
    - 중심 경도
    - 중심 위도

# 2. Naver Maps API

네이버 Maps API는 지도 제공, 주소 ↔ 좌표 변환(지오코딩), 경로 탐색 기능을 제공합니다.

## 2-1. Dynamic Map(지도를 불러와 그리는 기능)

**축약 설명**

웹/앱 화면에 움직이는 네이버 지도를 표시하고, 지도 위에 마커·선·도형 등을 표현합니다.

**제공 또는 사용 가능 정보**

- 지도 화면
- 지도 중심 좌표
- 지도 확대/축소 레벨
- 지도 타입
- 지도 이동/확대/축소 조작
- 마커 표시
- 정보창 표시
- 선 표시
- 도형 표시
- 지도 이벤트 정보
    - 클릭
    - 드래그
    - 줌 변경
    - 지도 영역 변경
- 지도 UI 컨트롤
- 지도 스타일 적용 정보

---

## 2-2. Static Map(지도를 불러와 그리는 기능)

**축약 설명**

지도 화면을 이미지 형태로 생성합니다.

**제공 또는 사용 가능 정보**

- 지도 이미지
- 지정 좌표 중심의 지도
- 지정 확대 레벨의 지도
- 지정 크기의 지도 이미지
- 마커가 포함된 지도 이미지
- 선 또는 경로가 포함된 지도 이미지
- 지도 타입이 적용된 이미지
