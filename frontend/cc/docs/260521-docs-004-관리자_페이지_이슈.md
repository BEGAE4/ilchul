## 💡 기능 설명

- 신고, 문의에 대한 관리자 기능을 갖춘 admin 페이지를 구현했습니다.
- 운영자가 `/admin` 경로에서 신고 큐를 조회·처리(제재)하고, 문의를 조회·답변할 수 있는 **반응형 웹 콘솔**을 제공합니다.
- 사용자 앱(`PageLayout`, 하단 탭)과 분리된 `AdminLayout`(사이드바·헤더)을 사용합니다.

---

## 📝 상세 설명

### 라우트 · 레이아웃

| 경로 | 설명 |
|------|------|
| `/admin` | 관리자 홈(신고 목록으로 진입) |
| `/admin/reports` | 신고 목록 (필터·페이지네이션) |
| `/admin/reports/[reportId]` | 신고 상세 · 제재 발급 |
| `/admin/inquiries` | 문의 목록 |
| `/admin/inquiries/[inquiryId]` | 문의 상세 · 답변 CRUD |

- `widgets/admin-layout`: `AdminLayout`, `AdminSidebar`, `AdminHeader`
- `shared/ui/admin`: `DataTable`, `Pagination`, `FilterChip`, `EmptyState`, `ConfirmDialog`

### 신고(Report) 관리

- 목록: 상태·대상 유형·사유 필터, PC 테이블 / 모바일 카드 뷰
- 상세: 신고 대상·신고자·사유·컨텍스트 URL 등 패널 표시
- 제재: `SanctionForm`으로 경고·일시 정지·영구 정지·콘텐츠 블라인드 등 발급 (mock BFF)
- API: `features/admin-report` + BFF `/api/admin/reports/*`

### 문의(Inquiry) 관리

- 목록: 상태·유형 필터, 테이블/카드 반응형
- 상세: 문의 본문·작성자 정보
- 답변: 등록·수정·삭제 (`AnswerForm`, `AnswerList`)
- API: `features/admin-inquiry` + BFF `/api/admin/inquiries/*`

### 기술·범위 메모

- **P0**: 리스트 / 상세 / 제재·답변 액션 (mock-first BFF)
- **P1 제외**: 통계 대시보드, 운영자 활동 로그, 일괄 처리
- 어드민 인증·역할 가드는 UX 수준; 실 권한 검증은 백엔드 연동 시 강화 예정
- API 명세 참고: [cc/api/v1/report.md](../api/v1/report.md)
