# CMT-34. 댓글 작성
# URL : POST api/reply/{planId}

**상의해봐야 할 것** : 

- 현재 대댓글 허용, 대댓글의 깊이는 1로 제한하고, 대신에 ‘@유저’의 형태로 유저를 언급하는 방식으로 구현할까 생각중.
- 대댓글 허용여부, 대댓글 깊이, ‘@유저’언급 기능 사용여부 → 결정해야 함

### 본문 파라미터

| **이름** | **유형** | **필수/선택** | **설명** |
| --- | --- | --- | --- |
| contents | String | 필수 | 댓글 본문 내용 |
| parent | Integer | 선택 | 부모댓글 아이디(일반 댓글이면 null |
|  |  |  |  |

### 응답 코드

| status | message |
| --- | --- |
| 200 | 성공 |
| 400 | 요청 데이터 오류 |

## 코드 예시

### 요청

```jsx
{
  "content": "대댓글 내용입니다.",
  "parentId": 123  // 부모 댓글의 ID (일반 댓글일 경우 null)
}
```

### 응답

```java
{
	status : 200
}
```
